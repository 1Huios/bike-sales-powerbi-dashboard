# Bike Sales Analysis (Power BI)

## Overview
This project focuses on analyzing bike sales data using Microsoft Power BI.
The dashboard provides insights into sales performance, customer patterns,
and product trends to support data-driven decision-making.

## Project Objective
- Understand overall bike sales performance
- Identify sales trends and patterns
- Analyze customer and product-level data
- Present insights through interactive visuals
